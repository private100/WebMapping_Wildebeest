Furui Sun's Diary

2015.05.20 2 hours

2015.06.02 2 hours

We had another discussion about the layout of the map. And Shuoyan came up with the design idea about the main map and Chiyuan mentioned that we should have 3 pages. I was responsible for the base map drawing task first.

2015.06.03 3 hours

I searched relevant materials: text and images, and download them with writing the attribution information in en text file.

2015.06.04 4 hours

I used the illustrator to draw the base map according to the reference I found, and made some changes to the design. I also drew the area where wildebeests are in different month.

2015.06.06 2 hours

I adjusted the color and the scale of the map to make it proper to be shown in the screen.

We had a discussion again to keep everything in order. Then I began to design the second page, the overview page. I cited the SVG elements from Shuoyan and added the second map, using Google API, to add some interactive part into our map. Then I added the navigation bar to the page with some responsive functions when the window is very small or narrow, especially in some mobile terminals.

There was some problem about the base map on different screens, so I had to rescale the base map and drew more about the background area. After this, I searched and downloaded relevant pictures for the main page’s information box, according to different months.

Chiyuan Gu's Diary

2015/6/2 (3h)

We discussed the design of layout and the type of map including interactivities and functionalities. After referring to some existing web map, we are going to use scroll to show the change of migration area according to different months. So this map will look like a story map. Our web map contains 3 pages which are the main map page, overview introduction page and the project diary page. I am assigned the task about realize the interactivity.

2015/6/6 (8h)

I start to do my part. In order to realize the functionality, I choose to use skrollr, a parallax scrolling JavaScript library. Using skrollr.min.js our map is able to move around the background map image and animate the movements of the animal activity areas on the map as the user scrolls down the page. Firstly, I learn something about skrollr.min.js following the tutorials on the Internet. Then I try to realize some basic effects when I scroll down the page. I create a draft web page which contains the background map and a text box. When I scroll down the text, you can see that background image can move and other images can change their size and change color.

2015/6/7 (2h)

I tell my members the difficulties I suffered from and exchange the suggestions about our map.

2015/6/8 (4h)

I get the image materials from my members and begin to fix them on the right position on the web page. Firstly, I just set the "top" "left" attributes. As a result, the image will move according to the size of my browser window. Then it takes me a lot of time to address this problem. Finally, I find use "margin-top" or "margin-left" can fixed the image position.

2015/6/10 (4h)

Today's task is to adjust the position of each image. It is a tough task because I can only do it by manual. For this part, I do a lot of repeat work which drives me crazy.

2015/6/12 (1h)

We work together to figure out some detailed problems of our map such as color, font and responsive design.

Shuoyan Huang's Diary

2015.06.02

Discussed the content and basic layout of the map and remaining pages. I was inspired by some story maps, especially this one, and told my team mates about the idea. Then we divided the work among us. Time spent: 2 hours.

2015.06.07

Had a discussion with team mates, and draw the timeline in Illustrator. (But we decided later that the timeline is unnecessary...) Time spent: 2 hours.

2015.06.11

Styled the text area and edited the text content to display the information month by month. Studied the skrollr library and set data-attribute to (almost) synchronize the highlighted areas and corresponding text when scrolling. Made the navigation bar responsive. Set two break points: 840px and 640 px. Added more content to the scrolling text area. Added image captions. Added more content to and styled the "About" page. Collected everyone's diary and added the project diary page.
Time spent: 8 hours.

2015.06.13

Adjusted the scrolling text description at the right hand side one last time to make sure they are aligned with the moving areas on the map.

Continued to work with the responsive aspect of the website, styled nav bar and content in different window sizes, and used jQuery to show/hide the dropdown menu at small screen size. (unfortunately you have to click the menu again to hide the dropdown, not by clicking elsewhere.)
